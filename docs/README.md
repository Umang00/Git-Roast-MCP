## Quick Start

For most users, the quickest path is:

1. Read [SETUP.md](SETUP.md) - Get it running locally
2. Test with Claude Desktop or Claude Code
3. (Optional) Read [DEPLOYMENT.md](DEPLOYMENT.md) - Deploy to the cloud
4. (If entering hackathon) Read [HACKATHON.md](HACKATHON.md) - Submit properly
